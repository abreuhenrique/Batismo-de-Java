# 🥷 JavaNinja

Bem-vindo ao **JavaNinja**! Este repositório foi criado para organizar, documentar e versionar meus estudos e práticas na linguagem **Java** que estou aprendendo com o Curso de Java do Fiasco.

## 🎯 Objetivo
O objetivo principal deste repositório é consolidar os conhecimentos adquiridos durante meus estudos de programação, servindo como um material de consulta rápida e um histórico do meu progresso na linguagem.

## 🗂️ Estrutura do Projeto
O projeto está dividido em pacotes de acordo com o nível de complexidade e os tópicos abordados:

### 🟢 Nível Básico
* Variáveis
* Estruturas Condicionais
* Estruturas de Repetição
* Scanner
* Operador Ternário
* Arrays
* *Resolução de Desafios Práticos*

### 🟡 Nível Intermediário
* **Orientação a Objetos:** Classes, Atributos, Métodos e Parâmetros
* **Herança:** Conceitos de Herança e Herança Múltipla
* **Polimorfismo e Abstração**
* **Construtores:** Sobrecarga de Construtores e Metodos (Overload)
* **Interfaces**
* **@Override**
* **Referencia de Memoria x Valor em memoria**
* **Desafio: Criar um Banco de Konoha -** Banco com Conta Corrente e Poupança que permite fazer depositos e tranferncias entre diferentes contas cadastradas no Banco.
* **Classes, Metodos e Atributos Final**
* **Encapsulamento:** Getters e Setters
* **ENUMS**
* **Lists**

## 🛠️ Tecnologias e Ferramentas Utilizadas
* **Linguagem:** Java
* **IDE:** IntelliJ IDEA
* **Versionamento:** Git e GitHub
